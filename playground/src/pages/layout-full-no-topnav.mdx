# Full Layout without Top Nav

This page uses the default `full` layout but hides the top navigation with `showTopNav: false`.

- ❌ No top navigation
- ✅ Sidebar
- ✅ Outline
- ✅ Content with footer

## Configuration

```yaml
---
showTopNav: false
---
```

## Use cases

Useful for embedded documentation or when you want a cleaner reading experience.
