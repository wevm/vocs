### Twoslash

Vocs supports [TypeScript Twoslash](https://www.typescriptlang.org/dev/twoslash/) syntax. [Read more](https://shikijs.github.io/twoslash/).

#### Preview

```ts twoslash
// @errors: 2540
interface Todo {
  title: string
}

const todo: Readonly<Todo> = {
  title: 'Delete inactive users'.toUpperCase(), // [!code hl]
//  ^?
}

todo.title = 'Hello'

Number.parseInt('123', 10)
//      ^|



```

## Headings
