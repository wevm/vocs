# Getting Started

Get up and running with Vocs in minutes.

## Installation

:::code-group

```bash [pnpm]
pnpm add vocs
```

```bash [npm]
npm install vocs
```

```bash [yarn]
yarn add vocs
```

:::

## Quick Start

1. Create a `vocs.config.ts` file in your project root
2. Add your first page in `src/pages/index.mdx`
3. Run the development server

```bash
pnpm vocs dev
```
