# Code & Syntax Highlighting [Rich markup and annotations for code]

Use fenced code blocks with triple backticks (`` ``` ``) to format code. Add a language identifier to enable syntax highlighting.

:::code-group
<div data-title="Preview">
```ts
export function hello(name: string = 'World') {
  return `Hello, ${name}!`
}
```
</div>
````md [Markdown]
```ts
export function hello(name: string = 'World') {
  return `Hello, ${name}!`
}
```
````
:::

## Titles

````md [Markdown]
// @\log: Custom log message

// @\error: Custom error message

// @\warn: Custom warning message

// @\annotate: Custom annotation message
````

## Markdown Snippets

Vocs supports reusable snippets inside fenced code blocks.

### Virtual Files

Define a virtual file with `filename="..."`, then include it in other code blocks.

:::code-group
<div data-title="Preview">
```ts filename="client.ts"
import { http, createPublicClient } from 'viem'
import { mainnet } from 'viem/chains'

export const client = createPublicClient({
  chain: mainnet,
  transport: http(),
})
```
</div>
````md [Markdown]
```ts filename="client.ts"
import { http, createPublicClient } from 'viem'
import { mainnet } from 'viem/chains'

export const client = createPublicClient({
  chain: mainnet,
  transport: http(),
})
```
````
:::

Now include the virtual file in another code block:

:::code-group
<div data-title="Preview">
```ts
// [!include client.ts]
const blockNumber = await client.getBlockNumber()
```
</div>
````md [Markdown]
```ts
// [\!include client.ts]
const blockNumber = await client.getBlockNumber()
```
````
:::

### Physical Files

Include code from files on disk with a root-relative `~/` path.

:::code-group
<div data-title="Preview">
```ts
// [!include ~/snippets/example.ts]
```
</div>
````md [Markdown]
```ts
// [\!include ~/snippets/example.ts]
```
````
:::

### Regions

Mark regions in a source file, then include only the part you need.

::::code-group
<div data-title="Preview">
```ts
// [!include ~/snippets/example.ts:setup]
```
</div>
``````md [Markdown]
```ts [src/snippets/example.ts]
// [!region setup]
import { createPublicClient, http } from 'viem'
import { mainnet } from 'viem/chains'

export const client = createPublicClient({
  chain: mainnet,
  transport: http(),
})
// [!endregion setup]
```

```md
```ts
// [\!include ~/snippets/example.ts:setup]
```
```
``````
::::

Modify included code on the fly with `/find/replace/` syntax.

</div>
